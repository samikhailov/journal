Редакционные правила для авторов

Все материалы, передаваемые для рассмотрения к публикации в редакцию журнала, необходимо оформлять в соответствии с редакционными правилами для авторов данного журнала. Вы можете найти их на странице с информацией о журнале. Электронная версия статьи должна быть оформлена в стилевом файле согласно издательским правилам для авторов.

В случае противоречий в оформлении статьи ориентируйтесь на редакционные правила.

ГДЕ НАЙТИ РЕДАКЦИОННЫЕ ПРАВИЛА ДЛЯ АВТОРОВ?

Шаг 1

Выберите в каталоге подходящий вам по тематике журнал.


Шаг 2

Перейдите по ссылке 'подробнее о журнале' на страницу с информацией о выбранном журнале.


Шаг 3

Если на странице журнала появилась вкладка 'Правила для авторов', перейдите в этот раздел. Все имеющиеся правила предоставляются в PDF формате. Если на странице отсутствует ссылка на файл для скачивания, или в меню журнала нет такой вкладки, попробуйте перейти на сайт с информацией об английской версии журнала и переключите язык интерфейса сайта. Возможно, есть аналог правил на другом языке.
